# HTTPS 적용 및 AWS 접속 가이드

이 폴더는 AWS 인스턴스(Bitnami LAMP) 환경 구축 및 관리와 관련된 설정, 접속 정보 등을 보관하기 위해 생성되었습니다.

## 🚀 진행된 주요 작업

1. **AWS EC2 SSH 접속 설정**
   - 발급된 보안 키를 활용하여 Bitnami 계정으로 서버에 안전하게 접근합니다.
   
2. **무료 SSL 인증서(Let's Encrypt) 발급 및 HTTPS 적용**
   - 적용 도메인: `jibinjibinjibin.shop`, `www.jibinjibinjibin.shop`
   - `bncert-tool`을 사용하여 무료 보안 인증서를 발급받았으며 매월 자동으로 갱신되도록 크론(Cron) 작업이 등록되어 있습니다.
   - 모든 사용자 요청(HTTP)이 보안 연결인 HTTPS로 자동 전환(리다이렉트)됩니다.

3. **로컬 개발 환경 설정 (macOS Apple Silicon)**
   - 원활한 소스코드 및 원격 저장소 관리를 위해 최신 버전의 GitHub CLI(`gh`)를 로컬 환경에 구축하고 경로 설정을 완료했습니다.

## 📝 파일 설명

- `AWS.rtf`: 서버 접속에 필요한 고유 접속 정보(IP, 계정 등)가 기재되어 있습니다. **주의: 보안을 위해 절대 GitHub 등의 외부에 업로드하거나 노출하지 마세요.**
